# Общее решение
Проект состоит из файлов на языке **Python** которые содержат функции для вычисления площадей и периметров некоторых геометрических фигур.

# Обзор функций
## Файл circle.py 
### `area(r)`:
```Python
def area(r):
    return math.pi * r * r
```
Принимает в качестве параметра радиус круга, возвращает его площадь, вычисляя ее по формуле `S = πr^2`, где π - число Пи, берется из библиотеки “math” библиотеки ЯП.

| Входные данные | Выходные данные |
|----------------|-----------------|
| 3              | 28.274333882308138|
| 4              | 50.26548245743669|
| 0              | 0.0             |

### `perimetr(r)`:
```Python
def perimeter(r):
    return 2 * math.pi * r
```
Принимает в качестве параметра радиус круга, возвращает длину его окружности, вычисляя ее по формуле `S = 2πr`, где π - число Пи, берется из библиотеки “math” библиотеки ЯП.

| Входные данные | Выходные данные |
|----------------|-----------------|
| 2              | 12.566370614359172|
| 5              | 31.41592653589793|
| 0              | 0.0             |

## Файл square.py
### `area(a)`:
```Python
def area(a):
    return a * a
```
Принимает в качестве параметра длину стороны квадрата, возвращает его площадь, вычисляя ее по формуле `S = r^2`.

| Входные данные | Выходные данные |
|----------------|-----------------|
| 2              | 4               |
| 3              | 9               |
| 1.5            | 2.25            |

### `perimetr(a)`:
```Python
def perimeter(a):
    return 4 * a
```
Принимает в качестве параметра длину стороны квадрата, возвращает его периметр, вычисляя его по формуле `S = 4r`.

| Входные данные | Выходные данные |
|----------------|-----------------|
| 3              | 12              |
| 4              | 16              |
| 5              | 20              |

# Unit тесты
## Для каждой вышеперечисленной функции есть тесты. 
### Чтобы запустить тесты нужно ввести в командную строку в корне проекта одну из следующих команд
* Тесты для квадрата: `python -m unittest tests/test_square`
* Тесты для Круга: `python -m unittest tests/test_circle`
* Запуск всех тестов: `python -m unittest tests/test*`

### Если все тесты прошли успешно выведится следующее сообщение:
```

OK
```
